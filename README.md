# Derpy-Bird
### A Flappy Bird Remake made using Unity and C# Scripting

### Gameplay
![Gameplay Loop of Derpy Bird](https://github.com/BigTimmyTom/Derpy-Bird/blob/main/Gameplay.gif)

### How to Play
- **USE SPACEBAR** to make Derpy Bird fly upwards
- **Avoid** the incoming poles
- Flying **too high** or **too low** might be *your last flap...*

### Features
- High Score Tracker
- Particle System to simulate Clouds
- Randomized Pipe Spawner Script
- Start Menu
  - Start Button to Start Your Flying Journey
  - Exit Button to Close Application
- Game Over Screen
  - Restart Button to Fly Again
  - Menu Button to Return to Menu
- Randomized Taunts after each Game
